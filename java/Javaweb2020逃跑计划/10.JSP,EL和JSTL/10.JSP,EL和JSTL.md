# 1. jsp
## jsp

- 1. jsp 是java的特殊html
- 2. 特殊的页面,里面可以写html, 也可以写java
- 3. 他的原理就是一个 `Servlet`
- 4. 他有三种脚本如下
  - 1. `<% %>`: 作用范围较小.  定义的java代码,在service方法中. service方法中可以定义什么, 这个脚本中就可以定义什么.
  - 2. `<%! %> `:可能会有线程问题. 作用范围大.  定义的java代码, 在jsp转换后的java类的成员位置.
  - 3. `<%= %>`: 输出用的
